2020-01-18 /
  • Linux
 / 点击 / 阅读耗时 1 分钟

问题描述：

用户在系统内进行创建文件的时候，出现空间不足提示：“No space left on device ⋯”

解决过程：

Linux 系统空间满，常见的原因包括：
（1）分区容量满；
（2）分区inode耗尽；
（3）僵尸文件：已删除文件因句柄被占用未释放导致相应空间未释放。
